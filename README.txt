Overview

Short application replicating Tetris game via display of Nokia 5510. 

Description

Project requires Discovery STM32F407VG board, Nokia 5510 display, set of external buttons and mini-USB cable. 

Tools

IDE used in the project is CooCox IDE. Code has been written in C using PCD8544 library found on https://stm32f4-discovery.net/

How to run

To run simple loading to Discovery board suffices.

How to compile

Compile as is. 

Future improvements

Subsequent versions will provide better pseudo number generator as well as include more block patterns to increase difficulty and variety.
